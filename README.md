### Installation

Install the dependencies and devDependencies and start the server.

```sh
cd bcd-onlineshop-server
npm i / yarn install
npm start / yarn start
```

### Used in project

- cors: ^2.8.5
- express: ^4.17.1
- mongoose: ^5.12.0
- nodemon: ^2.0.7

> I only used get methods because it was enough for the project
